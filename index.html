How to use PS5 Vault

Quick start

  1. Click "Browse" next to Source and choose the folder or drive that contains your game folders.
  2. Click SCAN. The app will locate validated game folders and list them with thumbnails.
  3. Select the entries you want to process using the checkboxes (use Select All / Unselect All to help).
  4. Pick a Destination where the organized folders should be created or moved to.
  5. Choose Action and Layout (these controls are under Destination). Action controls whether to create folders, copy, or move. Layout selects the destination folder structure.
  6. Click GO. A confirmation dialog (From â†’ To) appears. Confirm to start.
  7. During transfers, only the Cancel button is shown. Wait for completion.

Notes

  • If a target already exists you will be prompted (or default) to handle conflicts via skip or rename.
  • For fastest moves, place Destination on the same drive as Source and use Move. If drives differ, Copy+verify is used for safety.
  • The PPSA folder is removed, but internal game folder structure (e.g., sce_sys) is preserved.
